# vue-to-do-list

## プロジェクト概要
Vueの練習用に作成した簡単なto-do-listアプリです。基本操作、タスク追加、編集、削除、完了、未完了のトグル機能、優先順位フィルタリングを実装。

## Project setup
```
npm install
```
